## 网站内容

你是否想过，晚餐里的一个汉堡，也能成为你今天心情和选择的答案？在 Hamburguar，你可以亲手搭配六层食材，把日常口味变成专属卦象，再把你正在烦恼的问题交给它。每一次点击，都是一次轻松有趣的探索；每一份结果，都像一张写给当下自己的小纸条，陪你在忙碌里找到一点方向感与好心情。
